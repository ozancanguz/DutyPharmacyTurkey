# DutyPharmacyTurkey

*I have developed mobile app that list duty pharmacies of asking district in Turkey and KKTC. 

**Tools that I used** :

- Navigation Component
- Hilt for dependency injection
- Retrofit 
- Postman for Api testing.
- Livedata.
- View Binding
- Coroutines
- RESTApi
- LottieAnimationView
